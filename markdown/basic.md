# Markdown Basic
Markdown은 .md 확장자로 생성한다.


Heading(제목)
제목은 #으로 시작하며, 한칸 띄고 작성한다.
총6단계로 하위 제목을 분류할 수 있다.
평의상 H1~ H6 로 지칭하며, #의 갯수로 구분한다.

# H1(대제목)
## H2
### H3
#### H4
##### H5
###### H6


## paragraph (문단/내용)
글은 보통 쓰듯이 쓰면 된다.
하지만 엔터를 한번 치면, 줄바꿈이 되지 않는다. 말을 길게 하면 어느 순간 알아셔 내려 간다

두번 엔터를 쳐야, 실제 문단 구분이 일어난다.

- blod: * 두개
- Italic: * 한개
- code: ` 한개
- cancle: ~ 두개

## 블럭 강조

### Table(표)
|id|name|age|location|
|---|---|---|---|
|1|김김김|20|`서울`|
|2| **이이이** |25|부산|
|3| 박박박 |28 | 대전|

### code block(코드블럭)
백틱 3개로 감싸면 코드 블럭이 생성된다

```
$ cd ~
$ mkdir aaa
$ cd aaa
$ touch a.txt
$ rm a.txt
```


### Math Block (수식 불럭)
Latex(레이텍) 전문 표기법 학습 필요.
`$` 두개로 삼까서 블럭 새엇ㅇ.
$$
\mathbb{N} = \{ a \in \mathbb{Z} : a > 0 \}
$$
